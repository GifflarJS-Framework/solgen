Module

createVariableModel

A Factory for creating a variable statement object model (json).

Parameters:
Name Type Attributes Default Description
_type string

The type of the variable (string, uint, ...).
_name string

The name of the variable.
_scope string <optional>
 ""

The scope of the varible (public, private).
_setMethod string <optional>
 false

If you want to generate automatically the variable set function.
_value string <optional>
 ""

The value to be assigned to the variable after the creation. This value actually can receive an statement object too.
